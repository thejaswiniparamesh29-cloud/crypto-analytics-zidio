# Crypto Analytics Dashboard — Zidio Internship

Live App: [Click Here](https://your-app-link.streamlit.app)

Time Series Analysis with Cryptocurrency using ARIMA, Prophet, and interactive dashboards.

**Features:**
- Live BTC, ETH, BNB, SOL, ADA data
- Candlestick charts + Bollinger Bands + Moving Averages
- ARIMA & Prophet forecasting
- Volatility, Drawdown, RSI, Sharpe Ratio analysis
- Downloadable CSV data

**Tech Stack:** Python, Streamlit, Plotly, Prophet, Statsmodels, YFinance
